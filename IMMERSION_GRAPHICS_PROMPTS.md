# 🎨 AI Graphics Generation Prompts for SpeakEasy Immersion Scenarios

**Target AI Tools:** Gemini 2.5 Pro, Midjourney v6, DALL-E 3, Stable Diffusion XL

**Design Guidelines:**
- Style: Modern flat illustration, minimal detail
- Color: Vibrant but not oversaturated
- Characters: Diverse, friendly, simple facial features
- Composition: Clear focal point, uncluttered
- Size: 1200x800px landscape orientation
- Mood: Welcoming, non-threatening, culturally appropriate

---

## 📋 **PROMPT TEMPLATE STRUCTURE**

```
[Scene description], [art style], [characters], [setting details], [mood/lighting], [color palette], [technical specs]

Common suffixes to add:
--ar 3:2 --style raw --v 6 (for Midjourney)
High quality, professional illustration, clean composition (for DALL-E/Gemini)
```
