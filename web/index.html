üöÄ Quick Setup

Get started with SpeakEasy in just a few minutes

üîπ Step 1: Install Ollama

Download Ollama from the official website:

https://ollama.ai

üîπ Step 2: Start Ollama Server

Open your terminal and run:

ollama serve

Leave this running in the background

üîπ Step 3: Download a Language Model

In a new terminal window, choose one of these models:

For fast responses (recommended for beginners):

ollama pull llama3.1:8b

For better quality and advanced features:

ollama pull qwen2.5:72b

üîπ Step 4: Verify Installation

Check that everything is working:

curl http://localhost:11434/api/tags

You should see a JSON response with your installed models

üîπ Step 5: Launch SpeakEasy

Once Ollama is running, you're ready to start learning!
